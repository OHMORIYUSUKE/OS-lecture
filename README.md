# AWS-academy

## 資料の内容

EC2 を用いて簡単な WEB アプリケーションを公開するために必要なことを解説しています。
ハンズオンでは、EC2 で Apach, MySQL を起動し、PHP から MySQL のデータの操作を行います。EC2 を起動・設定、EC2 に接続から解説しています。
